# Neo4J PHP REST API client #

Very basic at the moment.

## Features ##

* Node creation
* Node loading by id
* Node delete
* Create relationship between two nodes
* List / filter relationships on a node

## Getting started ##

* Run it
* php demo.php

A PHP that has:

* curl
